The Project:

Page and Brin wanted to create an advanced search engine that would operate in a unique way. The already existing search engines would “simply return(ed) a list of Web sites ranked by how often a search phrase appeared in them” (Hosch).

Their project started under the name “Backrub”, in Page’s dorm room at Stanford. Brin and Page note that other search engines have difficulty filtering “junk results”, that “often wash out any results that a user is interested in” (Page and Brin). Their solution to this issue was to create a search engine would rank sites based off the number of links that were included on the site.

This algorithm would soon be named “PageRank” (Lowe), and their search engine “Backrub” would begin operation on Stanford’s private network.
